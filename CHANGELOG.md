# Changelog

## [1.3.10]

### 本次更新内容

- [进行中]把dps数据库重写了，解决加载过慢或加载不出来。

### 近期更新内容

- [水表]修复新副本普通不记录
- [水表]修复两个英文字母id无法正确识别
- [水表]修复玩家被误认为召唤物，从而导致显示不出角色名；
- [水表]重写召唤物相关逻辑，解决召唤物过远时不识别
- [水表]修复设备探测时的一个问题，会保留没有addresses的设备，并依次探测
- [水表]修复了偶尔重启水表后内存占用增加
- [水表]优化dps后端
- [伤害贡献度] 添加boss血量统计和伤害贡献度功能，现在默认显示伤害贡献度的百分比（可以在设置中切换成伤害占比）
- [UI]现在默认不会在鼠标悬停时显示技能数据，可以在设置，dps外观中打开。
- [UI]在主页设置中启用。
- [UI]重新调整部分UI，在设置中增加存储缓存、支持和赞助页面等。
- [UI]修复了登录注册页面的中文乱码
- [角色评分]现在已经回归部分功能，并且通过api请求相比之前速度更快（依赖蜂窝）。
- [技能修复]修复奶妈技能名称，以及剑星吸血剑影响职业判定。
